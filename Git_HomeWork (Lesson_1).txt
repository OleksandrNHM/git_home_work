1. Переходим в репозиторий, далее в роздел Code -> SSH, копируем ссылку. Далее идем в git-bash и копируем содержимое с помощу команды git clone + вставляем ссылку.

2.  git add <file>

3. git pull,  cat <file> 


4. git remote -v: Показывает URL-адреса для извлечения (fetch) и отправки (push) изменений в удалённый репозиторий.

git status: Эта команда показывает текущее состояние нашего рабочего каталога и индекса.
Она сообщает, есть ли изменения, которые не были зафиксированы (не добавлены в индекс) или не зафиксированы (не закоммичены).
Выводит информацию о неотслеживаемых файлах, изменениях в отслеживаемых файлах и другие статусы.

git log: Эта команда выводит историю коммитов в репозитории.
Показывает информацию о каждом коммите, такую как хэш коммита, автор, дата, и сообщение коммита.
